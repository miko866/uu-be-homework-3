## Local Dependencies

Version of npm - `8.xx.xx`

Version of node - `16.xx.xx`

Docker or Podman must be installed.

## Installation

### Local developing (localhost:4000)

1.  In server root folder run `docker-compose up` for MongoDB
2.  Rename `.env.example` to `.env`
3.  Run `$ npm i`
4.  `$ npm run serve`
